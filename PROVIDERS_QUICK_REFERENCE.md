## Download Priority Logic

```
1. Check user country code (e.g., "IN" for India)
2. Get top 5 providers for that country (sorted by priority)
3. If < 5 providers, add from fallback region
4. If still < 5, add from Global English providers
5. Download providers in priority order
```

## Quick Implementation

```javascript
// Simple implementation
const countryCode = getUserCountry(); // 'IN', 'US', etc.
const config = require('./regional-providers.json');

const region = config.regions[countryCode];
let providers = region 
  ? region.providers.sort((a, b) => b.priority - a.priority).slice(0, 5)
  : config.fallbackProviders.slice(0, 5);

console.log('Downloading providers:', providers.map(p => p.provider));
```

## Status Legend

- ✅ **Active** - Provider is enabled and working
- ⚠️ **Limited** - Limited content or availability
- ❌ **Disabled** - Provider is disabled in manifest

## Notes

- All providers have **stream validation** enabled (filters dead links)
- **Priority 10** = Best/Highest quality
- **Priority 1** = Backup/Lower quality
- Providers are sorted by priority (highest first)
- Maximum recommended: **5 providers per region**
- Fallback chain: Region → Fallback Region → Global English
